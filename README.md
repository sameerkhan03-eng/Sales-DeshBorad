# 📊 Sales Dashboard Analysis (Power BI Project)

---

# 🚀 Project Overview
This project presents an interactive **Sales Dashboard** built using **Power BI**, designed to provide actionable insights into sales performance, profit trends, and customer purchasing behavior.

The dashboard helps stakeholders quickly understand key business metrics and make data-driven decisions.

---

# 📌 Key Features

- 🔢 **Total Orders:** 1000  
- 💰 **Total Sales:** 2.6M  
- 📦 **Total Quantity Sold:** 2969  
- 📈 **Total Profit:** 535K  

---

# 📊 Dashboard Insights

## 📅 Sales & Profit Trends
- Monthly analysis shows fluctuations in sales and profit  
- Highest sales observed in **December**, indicating seasonal demand  

## 💳 Payment Mode Analysis
- **Online Payments (53%)** dominate over Cash (47%)  
- Indicates growing digital adoption among customers  

## 🛍️ Sales by Product
**Top-performing products:**
- Sugar  
- Soap  
- Bedsheet  

These products contribute the highest revenue share.

## 🧾 Sales by Category
- **Grocery:** 1336K (Highest)  
- **Personal Care:** 747K  
- **Household:** 564K  

## 📆 Daily Sales Trend
- Sales show variability across days  
- Peaks indicate promotional or high-demand periods

# 🎯 Filters & Interactivity

- 📆 Month-wise Filtering  
- 💳 Payment Mode Selection (Cash / Online)  

These filters allow users to drill down into specific insights.
